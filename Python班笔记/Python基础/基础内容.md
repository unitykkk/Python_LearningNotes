##### 序列类型

###### 1，列表：存储多个数据，每一个数据称之为元素，每个之间用，来隔开区分。（可变类型）

```python
#取一部分叫做切片，遵循左闭右开
In [1]: a = [1, 2, 3, 'bb', 'cc', 'dd']
In [2]: a[1:3]
Out[2]: [2, 3]
#索引取值：有序才可以索引取值，要在范围内取值
#切片取值：
[1:] 从当前索引位置取到最后
[1:3] 取索引值为1到3的元素
[1:3:2] 最后一个元素叫做步长
```

##### 拆包：

```python
In [3]: tu = (1, 2, 3, 4, 5)

In [4]: a, *b, c = tu

In [5]: a
Out[5]: 1

In [6]: b
Out[6]: [2, 3, 4]

In [7]: c
Out[7]: 5
    #   *代表多个，通配符
```
