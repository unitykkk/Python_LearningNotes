## Linux和Vim命令

### linux

```python
cd. 移动到当前目录
cd.. 移动到父目录
history 查看历史命令
! 数字 执行历史的命令
cd / 移动到根目录
cd 返回到当前的用户目录
/home 是家目录，家目录下的目录叫做用户目录
cd - 返回到刚才所在的目录
cd ~ 移动到家目录
绝对路径就是全路径。
相对路径就是当前目录下的路径，可以直接写要去的文件夹的名字。
ls -a 查看隐藏文件 文件名前带.就是隐藏文件
ls -a -l 查看隐藏文件的详细信息
ls -al 同上
ls p* 文件带p的都查出来
ls p? 代表只有一个p的
*代表多个字符，?代表只有一个
touch 创建文件
mkdir 创建文件夹
rm 删除文件
rmdir 删除文件夹
rm -r 删除文件夹，这个文件夹里还有文件时使用
cat 查看一个文件的内容
ls > a 创建一个叫a的文件，如果有a这个文件会先清空再创建
ls >> a 代表在文件末尾再追加
ll 查看当前目录下所有文件的详细信息
cp 你要复制的文件 目标文件：  复制文件的用法
mv 文件 文件夹：移动文件到指定目录或文件夹
ping 网址或者ip地址  查看有没有网
ps 查看进程
ps aux 查看所有进程
top 查看性能信息 q退出
ctrl+u：清空当前命令行
ctrl+l：清屏
echo '绝不做隔壁老王' > test02.txt  添加内容到文件里，没有此文件就创建一个
more test02.txt  查看此文件内容


当遇到这个问题时：
E: 无法获得锁 /var/lib/dpkg/lock - open (11: 资源暂时不可用)
E: 无法锁定管理目录(/var/lib/dpkg/)，是否有其他进程正占用它？
    
使用方法一
sudo rm /var/cache/apt/archives/lock  
sudo rm /var/lib/dpkg/lock
方法二
查看apt-get相关进程
ps -e | grep apt 

$sudo kill 进程数字

linux的ctrl键组合功能
1：+a

光标移到行首

2：+e

光标移到行尾

3：+b

光标向左移动一个字符位置

4：+f

光标向右移动一个字符位置

5：+u

删除光标之前的字符（不包括光标所在字符）

6：+k

删除光标之后的字符（包括光标所在字符）

7：+y

粘贴刚才删除的字符

8：+n

显示下一个命令

9：+p

显示上一个命令

10：+l

清屏

11：+r

搜索之前输入过的命令

12：+t

交换光标所在和前一个字符位置
```

### vim

```python
vim 进入编辑模式

末行模式：
:进入末行模式 w代表保存，记得加回车 q是退出或者直接 :wq退出，:q! 强制退出
:/ 查找在光标位置向后搜索，:? 向前搜索
在查找后 n向下查找，N向上查找
:%s///g 替换

命令模式：            
进来的时候默认是命令模式，或者 esc

插入模式：
都是在命令模式的前提下进入
i，从光标位置的前一个插入
a，光标位置的后一个插入
I，大写的I再命令模式下回到当前行最开头
A, 到行尾
o，在下方新插入一行并进入插入模式
O, 在上方
shift zz 退出，两次

h 左 
j 下 
k 上 
l 下

gg 到最开头
G 到最后一行开头

shift+6 到当前行的行首，非空格
shift+4 到最后面
0 到行首，无视非空格
程序员基本已经告别小键盘

yy 复制一行内容
p 粘贴
yw 复制当前行光标以后光标后的内容，有空格就断开，相当一个单词复制
dd 删除一整行
dw 删除光标位置的单词
x 删除光标所在的字符
s 删除光标内容后在当前位置插入
u 撤销上一次操作
contr+r 反撤销，撤销u的操作

v+上下左右，从光标位置开始对一整块进行操作
V+上下左右，从当前行对一整块进行操作
shift + > 往右缩进
shift + < 往左缩进

f6 变pep8规则
f5 运行
f2 打开目录结构
f3 换背景
f4 插入版权信息

python3 文件 直接运行
```
